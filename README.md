# 🏠 House Price Prediction using Linear Regression

A Machine Learning project that predicts house prices using the **California Housing Dataset** and the **Linear Regression** algorithm. This project demonstrates the complete machine learning workflow, including data loading, exploratory data analysis (EDA), feature selection, model training, prediction, evaluation, and visualization.

---

## 📌 Project Overview

The objective of this project is to build a regression model capable of predicting house prices based on various housing-related features.

The project covers:

- Data Loading
- Exploratory Data Analysis (EDA)
- Feature Selection
- Train-Test Split
- Model Training
- Prediction
- Model Evaluation
- Data Visualization

---

## 📂 Dataset

**Dataset:** California Housing Dataset

The dataset contains information about housing in California, including:

- Median Income
- House Age
- Average Rooms
- Average Bedrooms
- Population
- Average Occupancy
- Latitude
- Longitude

**Target Variable**

- House Price

## 📊 Machine Learning Workflow

1. Import Required Libraries
2. Load California Housing Dataset
3. Perform Exploratory Data Analysis (EDA)
4. Check Missing Values
5. Visualize Data using Histograms
6. Generate Correlation Heatmap
7. Select Features and Target Variable
8. Split Dataset into Training and Testing Sets
9. Train Linear Regression Model
10. Predict House Prices
11. Evaluate Model Performance
12. Visualize Results

---

## 📈 Evaluation Metrics

The model is evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
